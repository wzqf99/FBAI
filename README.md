# 📖 ai文创平台 - 前端（Vue3）

## 📌 项目简介
本项目是文创平台的前端部分，基于 Vue3 + Vite + Element Plus 开发，主要实现：
- AI 文章创作（调用后端接口）,局部内容优化重写(续写,扩写,简写,续写)
- 文章管理（增删查改）
- 热门话题管理
- 热门话题生成可用提示词
- 前端页面布局（Vue Router + Pinia）

## 🛠️ 技术栈
- Vue3 + Vite
- Vue Router
- Pinia（状态管理）
- Element Plus（UI 组件库）
- SSE（流式数据响应）
- tiptap (文章编辑)
